# 🛍️ Django E-commerce Platform

A complete e-commerce solution with both a traditional website and a REST API, built with Django and Django REST Framework.

## ✨ Features

### For Website Users
- Browse products with search and category filtering
- Add/remove items from shopping cart
- Update quantities in cart
- Secure checkout process
- Order history tracking
- User authentication (login/register)

### For API Users (Mobile Apps, React, Vue)
- Full REST API endpoints
- JWT authentication
- Product catalog access
- Cart management
- Order processing
- Stock management

### Admin Features
- Complete product management (CRUD)
- Order management
- Stock control
- Category management
- User management via Django Admin
